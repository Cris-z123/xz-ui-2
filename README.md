<h1 align="center">XZ UI</h1>

<div align="center">
这是我个人在使用vu3.x过程中制作的一个UI组件库，希望它可以对你有帮助
</div>

- 首页: https://rookie996-object.github.io/xz-ui-website/

## 环境和依赖

- node
- npm
- vite
- eslint
- vue 3.x 全家桶

## 项目下载和运行

- 拉取项目代码

```bash
git clone https://github.com/rookie996-object/xz-ui-2.git
cd xz-ui-2
```

- 安装依赖

```
npm install
```

- 开发模式运行

```
npm run dev
```

- 编译项目

```
npm run build
```
