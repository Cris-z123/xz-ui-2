## 环境和依赖

- node
- npm
- vite
- eslint
- vue 3.x 全家桶

## 项目下载和运行

- 拉取项目代码

```bash
git clone https://github.com/Cris-z123/xz-ui-2.git
cd xz-ui-2
```

- 安装依赖

```
npm install
```

- 开发模式运行

```
npm run dev
```

- 编译项目

```
npm run build
```
